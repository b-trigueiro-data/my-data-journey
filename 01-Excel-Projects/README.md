# Project 01: Grenoble Cost of Living Analysis

## 📊 Project Overview
As an aspiring Data Analyst based in Grenoble, I created this project to practice fundamental data organization and analysis skills. This project simulates a monthly budget for a student living in the city, focusing on categorizing expenses and identifying spending patterns.

### 🛠️ Skills Applied:
* **Data Structuring:** Creating a clean dataset from scratch with appropriate headers.
* **Formulas:** Using basic Excel functions for totals and specific category sums.
* **Pivot Tables:** Aggregating data to see totals by category (Rent, Food, Transport, Leisure).
* **Data Visualization:** Creating charts to communicate insights clearly.

### 📈 Key Insights:
* **Expense Distribution:** Analysis of how much of the budget is allocated to fixed vs. variable costs.
* **Local Context:** The data reflects average prices and categories found in the Grenoble area.
